# Torbrika Reader Journal (POC)

Pequeña prueba rápida creada con ayuda de IA (sin ser programador web) para simular un lector web destinado a Kobo Libra Colour. La app incluye pestañas de objetivos, reseñas, deseados y un apartado "Acerca" con opciones de backup. Uso estrictamente personal; el código no pretende ser producción ni está optimizado.
